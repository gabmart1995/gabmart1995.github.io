Portafolio Personal

Estudios Audiophone S.A

Rol: Programador Frontend.

Mi primer proyecto independiente a nivel profesional, aplicando tecnologías web, este proyecto se levanto en 2 fases, La primera fase es el sitio web de la organización. Con el objetivo de darse a conocer al público captando clientes y luego una aplicación administrativa en que los miembros puedan agendar y gestionar los días de servicios acordados con los clientes, generando el presupuesto correspondiente para posteriormente ser descargado y enviarse por correo electrónico.

Desarrollado en:

El diseño del sitio administrativo se utiliza la librería ReactJS para el manejo de las interfaces de usuario, transpilandose e integrandose con el sitio web, con acceso únicamente a administradores. Desde el Backend se acordó con el equipo que la lógica del modelo de negocio se mantiene separada en un API REST, siendo construido con el MicroFramework Lumen en PHP, utilizando el modelo de Modelo Vista Controlador.

Products App

Rol: Programador Frontend y Asesor a nivel de Backend.

Un proyecto muy interesante, sucedío cuando culminaba de realizar el entrenamiento de un miembro de mi equipo y se presentó la oportunidad, me dijo que tenia contáctos que necesitaban una aplicación administrativa para gestión y control de inventario de sus negocios, generando reportes y estadisticas de sus ventas. Pero esos negocios tienen problemas de conectividad a Internet y una aplicación web no era una opción, así con esa idea en mente y varias reuniones nace el concepto Products App.

Desarrollado en:

El diseño se orientó a crear una aplicación de escritorio, utilizando NodeJS y ElectronJS como núcleo central del software, conectándose una base de datos MySql o Mariadb dentro de su propia red interna. Utiliza tecnologías web para el desarrollo de sus interfaces. Se mantuvo el estilo básico para que el software pueda masificarse y usado en diferentes modelos de negocios y puede ser personalizado de acuerdo a los requerimientos del cliente y el equipo de desarrollo.

Equipo de desarrollo:

Optimmanage - Polpoo

Rol: Programador Frontend

Es conocido como el ecosistema completo de geolocalización, es una aplicación web que colabora de una forma excepcional con las actividades de logística y distribución de productos, centralizada en un solo sitio. Se comunica mediante diversas aplicaciones móviles especializadas para cada punto del ecosistema. Luego la informacion es cruzada y cotejada para las empresas que necesiten visualizar la información de los chóferes, puntos de entrega, ventas y clientes en sus diferentes Apps o generando reportes.

Desarrollado en:

Mi rol durante ese periodo de 3 años en colaboración fue la de actualizar las interfaces de la aplicación administrativa y móvil. Optimmanage, Miwigo-franquicias, Optimmanage-chofer y Miwigo-preparador, destinadas a los clientes, chóferes y las preparadores al momento de despachar las órdenes de entrega. El sitio cambio su aparencia y su nombre a Polpoo, pero su modelo de negocio sigue siguiendo el mismo.
